# Personal Profile Website

A simple static profile website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main page structure
- `styles.css` — layout and styling
- `script.js` — mobile navigation toggle

## Customize

- Replace `Your Name` and the placeholder text with your own details.
- Update the `mailto:` link in the contact section.
- Add your own project descriptions or social links.

## Run locally

Open `index.html` in a browser or use a local server for development.
